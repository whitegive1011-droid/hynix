# AI Investment Operating System (AIOS)

AIOS is a lightweight personal investment decision support system.

Current status:

- Milestone 1: project skeleton and configuration.

Run locally:

```bash
python3 main.py
```

Run tests:

```bash
python3 -m pytest
```

The target command in the product prompt is `python main.py`. On this Mac, the available interpreter is `python3`, so the milestone gate is verified with `python3 main.py`.
